# SmartFollow AI

An automated payment follow-up tool built with Python, Streamlit, and Pandas.

Features:
--Upload roster and stakeholder files
--Automatic data merging and validation
--Identify overdue payments based on aging criteria
--Interactive dashboard for payment insights
--One-click follow-up email generation
--Sample data mode for demonstrations

Key Learnings:
--Used @st.cache_data to optimize performance and avoid unnecessary file reloads.
--Implemented st.stop() to safely terminate execution and handle missing inputs gracefully.
--Applied Pandas for data transformation, aggregation, and date processing.
--Built an end-to-end workflow to automate a previously manual process.

Impact:
Reduced a manual process that took approximately 2 hours to less than 2 minutes.
